## 初始化 `./start/`

在 [**《构建 Vue3 组件库 - 如何开始》**](/posts/setup-vue3-component-library/how-to-start) 中，调试目录其实是一个通过 Vite 初始化的 `SPA` 项目。

调试组件库，需要启动项目，在 `package.json` 中增加启动命令：

```diff:package.json
{
  "scripts": {
+    "start": "vite -c ./start/vite.config.ts"
  }
}
```
